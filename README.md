# Daybook

A personal journal and task management application.

## Features

- Daily journal entries
- Task management
- Calendar view
- User authentication

## Tech Stack

- React + TypeScript
- Vite
- Convex (backend)
- Tailwind CSS

## Setup

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Start development:
   ```bash
   pnpm dev
   ```

## Scripts

- `pnpm dev` - Start both frontend and backend
- `pnpm build` - Build for production
- `pnpm lint` - Run linting
